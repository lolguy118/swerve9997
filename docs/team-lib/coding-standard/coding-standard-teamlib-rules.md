# Team271-Lib Coding Rules (`CODE-LIB-*`) (Retired)

The reusable-library rules `CODE-LIB-001..007` are now part of the
adopted coding standard. The generic rules live in
[`../../coding-standard/java/Standard-Library.md`](../../coding-standard/java/Standard-Library.md);
the Team271-Lib concrete bindings (TObj dependency injection, centralized
CAN refresh, desired/applied state, fault isolation, mandatory timeouts,
the TObj lifecycle, and `ConstantsLib` isolation) live in the
[supplement](../../coding-standard/team271-lib-supplement.md). The
common-rule to library-API binding table remains in
[`coding-standard-library-notes.md`](coding-standard-library-notes.md).

Rule IDs are unchanged; the full retired text is in git history.
